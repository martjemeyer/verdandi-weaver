Verdandi Design System

A small, calm system for a contemplative site. Tokens live in tokens.css; components in components.css. Edit the tokens to retheme the entire site.

Foundation

Tone

Quiet, grounded, warm

Generous whitespace, serif display for warmth, sans for clarity, hairline rules. Imagery is the loud element — composition stays calm.

Voice

Plain, present, kind

Short sentences. No jargon. Use "you," not "the user." Lowercase except for proper nouns and the start of a sentence.

Typography

Eyebrow / overline

Display H1 — Cormorant

Heading H2 — Cormorant

Heading H3 — Cormorant 500

Heading H4 — Inter 500

Lead — used under titles. Slightly larger and softer in color than body, snug line-height.

Body — Inter 16/27. The default reading size. Long passages should sit in a narrow container (~70ch) for comfortable measure.

Meta · timestamps, footnotes, captions.

“Quote — Cormorant italic, used sparingly for emphasis.”
